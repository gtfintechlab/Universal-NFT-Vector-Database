# Useful Commands

## Setup this Directory

- Get Windows Subsystem for Linux (Or any Linux Distro)

```
python3.8 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

## Start Milvus

```
sudo ./start_milvus.sh "start"
```

## End Milvus

```
sudo ./start_milvus.sh "end"
```
